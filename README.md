Commands
/rq - Open Config

-Config Options as commands-
/rq add <name> <qty> - Add or update item
/rq remove <name> - Remove Item
/rq list - List all Items
/rq enable / disable - Toggle Auto-Buy feature
/rq verbose - Toggle Purchase Messages


After configuring items, upon speaking to a vendor who sells those items will automatically restock you up to the desired amount. 
If items are sold in batches, you will purchase until you have at least the quantity specified
