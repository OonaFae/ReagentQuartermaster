# Commands
<p>/rq - Open Config</p>
<p>-Config Options as commands- </p>
<p>/rq add <name> <qty> - Add or update item</p>
<p>/rq remove <name> - Remove Item</p>
<p>/rq list - List all Items</p>
<p>/rq enable / disable - Toggle Auto-Buy feature</p>
<p>/rq verbose - Toggle Purchase Messages</p>
<br></br>
<p>After configuring items, upon speaking to a vendor who sells those items will automatically restock you up to the desired amount. </p>
<p>If items are sold in batches, you will purchase until you have at least the quantity specified</p>
